# ASD Automatic Reinforcement Automation Tool

An engineering automation tool developed to streamline, optimize, and automate structural reinforcement detailing and rebar distribution inside AutoCAD Structural Detailing (ASD).

## 🚀 Features

* **Automated Rebar Distribution:** Automatically calculates and distributes reinforcement bars along structural elements.
* **Cross-Section Generation:** Draws accurate structural cross-sections and detailing dynamically.
* **Workflow Optimization:** Reduces manual drafting time and minimizes human errors in technical offices.
* **Intelligent Coordination:** Built with precise handling of coordinate systems and indexing logic.

## 🛠️ Built With

* **Language:** Python / AutoLISP *(Choose or keep both based on your stack)*
* **GUI & Automation:** PyAutoGUI / Automatic Mouse and Keyboard
